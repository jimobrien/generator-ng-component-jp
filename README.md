> [Yeoman](http://yeoman.io) generator


## Getting Started

### Yeoman Generators

To install generator-ng-component-jp from npm, run:

```
$ npm install -g generator-ng-component-jp
```

Finally, initiate the generator:

```
$ yo ng-component
```
